Bootstrap 5

Bootstrap 5 adalah versi terbaru dari Bootstrap, sebuah framework front-end open-source yang sangat populer untuk membangun situs web responsif dan aplikasi web. Bootstrap dikembangkan oleh Twitter dan telah menjadi salah satu alat yang paling banyak digunakan dalam pengembangan web karena menyediakan komponen dan gaya pradesain yang dapat mempercepat proses pembuatan situs web yang menarik dan responsif.

Tip: Add a white text color to all links in the navbar with the .navbar-dark class, or use the .navbar-light class to add a black text color.
